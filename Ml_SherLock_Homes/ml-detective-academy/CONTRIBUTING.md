# Contributing

Thanks for your interest in contributing to ML Detective Academy! Please follow these guidelines:

- Open an issue first to discuss major changes or new case proposals.
- Fork the repository and create a feature branch named `feature/your-topic`.
- Add tests for any new behavior.
- Keep changes small and focused; write clear commit messages.
- Run `black` and `flake8` (if available) before submitting a PR.

We appreciate your help! If you have questions, open an issue and tag a maintainer.
